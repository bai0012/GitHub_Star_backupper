## 如何使用

### 使用源代码程序

首先安装python，已在python3.10.11上测试

拉取该repo，并且安装Python依赖项：

```git
git clone https://github.com/bai0012/GitHub_Star_backupper
```

```Powershell
cd GitHub_Star_backupper
```

```python
pip install -r requirements.txt 
```
修改main.py中Github用户名和个人访问令牌及目标地址

如：

```python


```


使终端处于项目文件夹的基础上，运行

```Powershell

```
